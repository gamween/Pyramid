Tellement-French — Full Architecture

Everything runs on Devnet (lending + trading). Groth5 for ZK privacy. No SC needed.

What the app is (user perspective)

Tellement-French = a DeFi platform where you can:

1. EARN (Lending)
Deposit XRP/tokens into a Vault → earn yield from loan interest. Like a savings account but on-chain.
2. BORROW (Lending)
Take a loan from the Vault liquidity. Fixed term, fixed rate. Use borrowed funds to trade.
3. TRADE (Advanced Orders)
Set stop-loss, take-profit, trailing stop, OCO. A bot watches the DEX price and executes for you.
4. ACCUMULATE (DCA/TWAP)
Pre-sign N buy orders with Tickets. Bot submits them one by one at intervals. All on devnet.

Which XRPL tech does what

Feature XRPL Native Primitive Network How it works
Vault (deposit/earn) VaultCreate, VaultDeposit, VaultWithdraw Devnet XLS-65. User deposits, gets share MPTokens. Yield from loan interest.
Loans (borrow/repay) LoanBrokerSet, LoanSet, LoanPay, LoanManage Devnet XLS-66. Cosigned by broker + borrower. Fixed term + rate. Grace period + default.
Stop-Loss / Take-Profit EscrowCreate + EscrowFinish + OfferCreate Devnet User locks funds in Escrow (with crypto-condition). Watcher holds preimage. When price triggers → watcher finishes escrow + places DEX order.
Trailing Stop EscrowCreate + EscrowFinish + OfferCreate Devnet Same as SL/TP. Watcher tracks high watermark in memory. Trigger = price drops X% from high.
OCO 2x EscrowCreate (same CancelAfter) Devnet Two escrows with same expiry. When one triggers, the other expires naturally via CancelAfter.
DCA TicketCreate + N × pre-signed OfferCreate Devnet User creates N Tickets, pre-signs N OfferCreate txs. Watcher submits one at each interval. All on devnet — no Xahau needed.
TWAP TicketCreate + N × pre-signed OfferCreate Devnet Same as DCA but shorter intervals, larger total amount split into slices.
Private Orders (ZK) EscrowCreate + RISC0 proof + EscrowFinish Groth5 Trigger price hidden as hash commitment. Watcher generates ZK proof via Boundless. Escrow verifies on-chain.

Simplified Architecture

Devnet (everything core)
  • Lending: VaultCreate/Deposit/Withdraw
  • Borrowing: LoanBrokerSet/LoanSet/LoanPay
  • Orders: EscrowCreate/Finish + OfferCreate
  • DCA/TWAP: TicketCreate + pre-signed OfferCreate
  • Prices: book_offers + amm_info (native)
Groth5 (privacy only)
  • Private orders: EscrowCreate + ZK proof
  • Boundless prover network
  • On-chain verification
Xahau dropped. DCA/TWAP now uses Tickets on devnet. One less network to manage. Architecture is now 2 networks instead of 3.
