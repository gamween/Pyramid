# Legacy Watcher

`apps/watcher` is retained as legacy hackathon-era code.

It is not part of Pyramid's active V1 architecture.

Current canonical direction:

- frontend-only
- direct XRPL reads from the browser
- wallet signing in the frontend
- on-chain persistence instead of watcher-managed state

Do not treat this watcher service as the source of truth for the current product direction.
